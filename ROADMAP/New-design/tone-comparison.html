How to decide:
  • Tone A (Peach Pup): if "family + home" is your #1 anchor — grounded, photo-ready, Pinterest-mom vibe
  • Tone B (Bubble Gum): if "funny + cute" is your #1 anchor — sticker-pack energy, LINE-native, shareable
  • Tone C (Honey Paw): if "cozy + Thai warmth" is your #1 anchor — premium trust, long-term family, richness
  • Tone D1 (POPS Soft) — BEFORE: pink-dominant, softened POPS palette. Warm but reads feminine. Kept for comparison.
  • Tone D2 (POPS Balanced) — AFTER: neutral stone base, POPS tri-color used only for accents. Unisex — works for both male and female pet parents.

D1 → D2 rebalance notes:
• Background: pink-tinted #FFF4E8 → warm stone #F5F1EA
• Primary button: pink→coral gradient → coral→amber gradient (less sweet, more confident)
• Text heading: warm plum #4A2040 → warm charcoal #2E2A2E
• Pills/tags: pink backgrounds → neutral stone backgrounds
• Status bar gradient: butter→coral → amber→coral (dropped yellow-pink lean)
• POPS tri-color: used everywhere → accent only (avatars + active pet selector)
→ Same V6 bubble layout, same POPS brand DNA, different emotional register.
